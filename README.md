

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the Relavoux.

Its an E-commerce website, backed with shopify storefront api,


TechStack Used
NextJs
GraphQl
Tailwindcss
ContextApi
React
